How I made it.

Compatibily.

All major browser. IE9+.



How I managed scrollbars.

Negative left/top margins do not add scrollbars, it's tricky, but it works.



How I fixed ghost margins at the bottom without clearfix.

it should be a pain determinate which cell/cells will be on the very bottom of the row if you have responsive column.
Solution: With the last row you can add a bottom negative margin, so you can solve it nicely and easily without using javascript to remove bottom paddings from each cell.

Why I made it.

No JavasScript.

No Javascript/jQuery needed.



Containers.

You can set you own containers. Your container = your own width = your own specific design.



Responsive and FLUID, not fixed.

Why on Earth do we need something fixed. Is this responsive design? No.



px, rem, em, %... what else. Your own grids gutters.

You need to make a gallery? You NEED to have the FREEDOM to decide HOW thick gutters MUST be, using the same grid system.



How many colum you want.

I will not talk about the same thing: "12 column grids layout". You can set how many columns the grid system should have.



@media size limits.

You can customize them with your own personal taste.



Stop using ONE row.

You can add many column as you want. Imagine that you have a table with superpowers.



You will simply love it.

You can start making your personal container, your design, and that's it.
